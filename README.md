**SAGE-Perú** es una plataforma web integral diseñada para modernizar y digitalizar los procesos académicos 
y administrativos del **Centro Escolar "República del Perú"**. Este proyecto busca optimizar 
el registro de asistencia, calificaciones y la comunicación entre la institución y los padres de familia.

##  Arquitectura de Contenedores
El proyecto utiliza una arquitectura basada en microservicios empaquetados en **Docker**. El archivo `docker-compose.yml` (próximamente) orquestará los siguientes servicios:
1. **Web-App:** Servidor Apache con módulo de PHP.
2. **Database:** Motor MySQL para persistencia de datos escolares.
3. **K8s Manifests:** Archivos de configuración para despliegue escalable en clúster.
